Privacy Policy

Last updated: 2025-08-14

This site hosts utility pages used to integrate TikTok Ads data into Google Sheets. The site itself does not collect personal data, set cookies, or run analytics.

Data we may process

  • OAuth redirect parameters from TikTok (e.g., code, state) displayed on the callback page so the owner can copy them into Google Apps Script.

No data is stored on this site. Any access tokens are generated and stored by the owner in Google Apps Script properties.
